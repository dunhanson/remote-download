# Claude Code 项目配置

## docs 目录结构

```
docs/
├── knowledge/      # 知识文档（技术介绍、学习笔记等）
├── requirements/   # 需求文档（功能需求、用户故事等）
└── plan/           # 计划文档（实现计划、任务拆解等）
```

## 文档命名规则

文件名格式：`YYYY_MMDD_HHMM_主题.md`

示例：
- `2026_0327_2141_bullmq.md`
- `2026_0327_2141_需求说明.md`
- `2026_0327_2141_实现计划.md`

组成部分：
- `YYYY` - 年份（4位）
- `MM` - 月份（2位）
- `DD` - 日期（2位）
- `HHMM` - 时间（4位，24小时制）
- `主题` - 中文主题名称，用下划线分隔
